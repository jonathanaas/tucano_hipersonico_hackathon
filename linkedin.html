Sabe aquela planilha de conciliação que o time financeiro passa horas preenchendo? A gente acabou com ela em um hackathon. 😅 Apresentamos o Onfly Conciliador — built no Hackathon Onfly 2026 pelo time Tucano Hipersônico 🦜 O fluxo é absurdamente simples: 1️⃣ Sobe a fatura Pronto. 🎉 O sistema busca automaticamente os dados da Onfly via API e faz a conciliação na hora — sem segunda fonte manual, sem configuração, sem esforço. O motor cruza tudo, classifica cada registro e mostra exatamente onde está a divergência — com o valor da diferença na cara dura. Tecnologia usada: React + Vite + PapaParse + Onfly API. Leve, rápido, integrado. Orgulho demais desse projeto. 💚
